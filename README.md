# JESDX Personal Web

Version 1.0 of a personal web surface for JESDX. The current release is a
public-facing landing page and portfolio foundation, designed to evolve into a
personal ERP / operating system over later versions.

## Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Oxlint

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Deployment

Recommended Cloudflare Pages settings:

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`

## Roadmap

- v1.x: portfolio content, SEO, image optimization
- v2.x: system overview and dashboard routes
- v3.x: personal ERP modules such as projects, notes, CRM, and finance
